### Let’s turn **Good-Behavior** into a “spin-worthy” reward

Your backend is already ready for it (`assign_bonus_spin: true → daily_bonus row`).
All we need is a tiny UX tweak plus one handler on the client.

---

## 1 ⃣  Add the toggle to **GoodBehaviorDialog**

```tsx
// client/src/components/good-behavior-dialog.tsx
const [tickets, setTickets] = useState(1);
+const [giveSpin, setGiveSpin] = useState(false);

…

<Form>
  {/* existing ticket input */}
- <NumberInput value={tickets} onChange={setTickets} … />

+ <div className="flex items-center gap-2 mt-4">
+   <Switch
+     id="giveSpin"
+     checked={giveSpin}
+     onCheckedChange={setGiveSpin}
+   />
+   <label htmlFor="giveSpin" className="text-sm select-none">
+     Award a <strong>Bonus Wheel Spin</strong> instead of tickets
+   </label>
+ </div>

+ {/* Disable / fade the ticket box when spin is chosen */}
+ <NumberInput
+   value={tickets}
+   onChange={setTickets}
+   disabled={giveSpin}
+   className={cn(giveSpin && "opacity-40")}
+ />
```

*(Switch is from shadcn/ui; use any toggle you already prefer.)*

---

## 2 ⃣  Submit the correct payload

```tsx
const handleSubmit = async () => {
  const payload = {
    user_id: selectedChildId,
    reason,
-   tickets,
+   tickets: giveSpin ? 0 : tickets,
+   assign_bonus_spin: giveSpin,
  };

  const res = await apiRequest("/api/good-behavior", {
    method: "POST",
    body: JSON.stringify(payload),
  });

  // If the server created a bonus spin, pop the prompt:
+ if (res.bonus_triggered && res.daily_bonus_id) {
+   openSpinPromptModal({
+     bonusId: res.daily_bonus_id,
+     childName: child.name,
+     friendlyTrigger: "Good Behavior",
+   });
+ }
  …
};
```

## 3 ⃣  (Optional) tidy up validation

## 4 ⃣  No backend change needed ✅

The `POST /api/good-behavior` route you fixed earlier already:

* accepts `assign_bonus_spin`
* creates a `daily_bonus` row (`trigger_type: 'good_behavior_reward', is_spun: false`)
* returns `{ bonus_triggered: true, daily_bonus_id: … }`
